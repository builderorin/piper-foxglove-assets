# piper-foxglove-assets

This repository hosts public Foxglove robot-model assets for Piper 3D replay.

Contents:
- `urdf/piper-foxglove.urdf` — public replay URDF with raw GitHub mesh URLs
- `meshes/*.STL` — minimal mesh set referenced by the URDF
- `layouts/piper-arm-layout.json` — Foxglove layout copied from `jetson-organize`

Primary public URLs:
- URDF: `https://raw.githubusercontent.com/builderorin/piper-foxglove-assets/main/urdf/piper-foxglove.urdf`
- Layout: `https://raw.githubusercontent.com/builderorin/piper-foxglove-assets/main/layouts/piper-arm-layout.json`

This repository intentionally contains only public visualization assets and does not include private runtime code.
